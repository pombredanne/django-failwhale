Requirements
============

python >= 2.4

django >= 1.0

python-twitter >= 0.5


Installation
============

To install run

    ``python setup.py install``

which will install the application into python's site-packages directory.


Quick Setup
===========


settings.py
-----------

Add to INSTALLED_APPS:

	``failwhale``
